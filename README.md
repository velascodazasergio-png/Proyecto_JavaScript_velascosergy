# 🏗️ CampusBuild — Project Manager System

## 📄 Descripción del Proyecto

**CampusBuild** es una empresa líder en el sector de la construcción que busca optimizar la gestión de sus proyectos mediante una aplicación web moderna e intuitiva.

El sistema permitirá administrar proyectos en todas sus etapas, desde la planificación hasta la ejecución y entrega final. La plataforma centraliza la información clave, facilitando la colaboración entre equipos, el seguimiento del progreso y la toma de decisiones estratégicas.

La interfaz estará diseñada con un enfoque moderno (UI/UX), fluido y fácil de usar, garantizando una experiencia eficiente para todos los usuarios.

## 🛠️ Tecnologías Utilizadas

* **HTML**
  Uso de etiquetas como `<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer>`

* **CSS**
  Diseño completamente personalizado sin frameworks externos

* **CSS Flexbox & Grid**
  Para layouts responsivos y organización eficiente

* **JavaScript**
  Lógica de la aplicación y manipulación del DOM

* **Web Components**
  Creación de componentes reutilizables y modulares

* **LocalStorage**
  Persistencia de datos en el navegador

* **Diseño Responsivo**
  Adaptable desde dispositivos móviles hasta escritorio

## ⚙️ Funcionalidades Principales

### 📁 Gestión de Proyectos

* Crear, editar y eliminar proyectos
* Definir nombre, descripción y fechas

### 📋 Gestión de Actividades

* Registrar actividades con:

  * Responsable
  * Fecha de inicio
  * Duración
  * Estado (Pendiente, En proceso, Terminada)
* Editar y eliminar actividades

### 🎯 Gestión de Hitos

* Crear hitos clave del proyecto
* Asociarlos a actividades
* Seguimiento automático (se completan al finalizar actividades)

### 👷 Gestión de Recursos Humanos

* Registrar personal con:

  * Identificación
  * Nombre
  * Fecha de nacimiento
  * Tipo de sangre
  * ARL
  * Salario
  * Rol (ingeniero, supervisor, etc.)
* Asignar recursos a actividades

### 📅 Cronograma

* Visualización tipo calendario
* Seguimiento del estado de actividades por proyecto

---

## 🧠 Conceptos Clave

* **Actividad:** tarea específica dentro del proyecto
* **Hito:** punto importante que marca progreso
* **Recurso humano:** persona asignada a tareas
* **Calendario:** vista organizada del cronograma
